Java基础

Java环境配置

jdk

版本 1.8

  • jdk： Java Development Kit 是 Java 语言的软件开发工具包
  • jre： Java Runtime Environment 指Java运行环境

安装注意事项

  • 安装路径不要有中文和空格
  • JRE不要安在JDK目录下面

常用快捷键

  • win+d 回到桌面
  • win+e 打开文件资源管理器
  • win+r 打开运行窗口

环境变量

  1. 我的电脑 -> 右键 -> 属性 -> 高级系统设置 -> 高级 -> 环境变量
  2. 系统变量 -> 新建 -> 变量名: JAVA_HOME 变量值: Jdk安装路径
  3. 系统变量 -> Path ->编辑 -> 新建 %JAVA_HOME%\bin -> 新建 %JAVA_HOME%\jre\bin

测试

  1. win+r
  2. 输入cmd
  3. 输入java -version 或 java 或 javac

注意： 每修改一次环境变量，需要重新打开一个窗口进行测试
##IDE

IDEA

安装包及详细安装步骤：参考U盘或者网盘

JVM java虚拟机

java文件 -> 编译生成class文件（字节码文件，二进制文件）-> 加载到JVM中运行

JVM

  • Windows
  • Mac
  • Linux

每个操作系统安装JVM后，都可以加载class文件执行，Java具有跨平台特性，一次编译，多处运行

GC

垃圾回收， java垃圾回收机制：自动回收

Student stu = new Student();  创建一个对象，需要分配空间

stu=null; 回收的前提 这样之后会根据JVM内部的垃圾回收算法，当达到一定条件开始回收这些资源

面向对象之后，不用的对象，清空 设置为null

IDEA开发

  • 前端 ：webstorm
  • python：pycharme
  • java：Idea

src：源代码目录

在学习多线程之前，目前我们的代码都是单线程的，代码运行在主线程（main thread）中，执行顺序从上向

快捷键

  • psvm

    public static void main(String[] args) {      
     }   //main方法是程序执行的入口

  • sout

System.out.println();

  • ctrl + shift + /

    • 选中代码注释掉
    • 不选中代码的话是直接生成多行注释

  • /** + 回车

    /**
     * 直接生成文档注释
     */

  • ctrl + alt + L

    代码格式化

  • ctrl+shift+alt+u

  • ctrl+e

    最近的工作目录

改编码

  • GBK 国际码
  • UTF-8 万国码 国际通用编码

编码不一致会导致乱码

File -> Settings -> 输入Encod -> File Encodings -> 都改成UTF-8

注释

单行 //
多行 /* */ 快捷键：ctrl+shift+/
##包
package

Java项目 -> java文件成千上万个 -> 想找某一个文件很困难 -> 分包管理

后期

  • service
  • dao
  • pojo/bean
  • contraller

每个包中放跟这个包名关系密切的相关文件
命名: 域名倒置

baidu.com -> com.baidu

命名规范

公司.部门.项目名.项目模块
