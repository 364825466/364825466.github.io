HEXO博客的搭建

2019-08-10 617 字
HEXO

Hexo

搭建

  1. 安装git

  2. 安装node.js

  3. 创建blog文件夹，git bash打开

  4. 安装hexo

    • npm install -g hexo-cli

    • 查看版本

      hexo -v

    • 初始化hexo

      hexo init myblog //这个myblog起什么名字都行

    • cd myblog

    • npm install

    • hexo g

    • hexo s //打开hexo的服务

    在浏览器输入localhost:4000就可以看到你生成的博客了。

  5. 部署到github

    在github建一个仓库 用户名.github.io

    • 安装插件

      cnpm install --save hexo-deployer-git

    • 修改config文件

    deploy:   
      type: git   
      repo:https://github.com/YourgithubName/YourgithubName.github.io.git   
      branch: master

    • 部署到远端

      hexo d

写博客

  • hexo n "标题"

    然后就markdown写

    <!--more--> 标签下面的内容不显示

  • 插入图片：

    1. 设置站点配置_config.yml:将post_asset_folder: false改为post_asset_folder: true
    2. 安装插件npm install https://github.com/CodeFalling/hexo-asset-image -- save
    3. 运行hexo n “XXXXXX”,生成XXXXX.md博文时就会在/source/_posts目录下生成XXXXXX的文件夹，将你想在XXXXX博文中插入的照片放置到这个同名文件夹中即可，图片的命名随意。
    4. 添加图片:在想添加的位置写入![你想输入的替代文字](/xxxx/图片名.jpg)（斜杠都不能少）

  • 发表

    在bolg目录

    • hexo clean //清理一下
    • hexo g //生成
    • hexo s

  • 修改主题

    github.com/litten/hexo-theme-yilia

    • 下载主题

      git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

    • 在config文件修改

    ​ theme 后面改成你的主题名

  • hexo clean 清理

    hexo g 生成

    hexo s 发布

    localhost4000端口改了

  • hexo d 推到远端 github的就改了

自定义配置

打开根目录config（不要用记事本，用vscode）

参数 含义
title 网站的标题
subtitle 网站的副标题
description 网站的描述
author 你的名字
language 网站使用的语言(中文：zh-CN)
timezone 时区，默认电脑时区
theme 主题

  • 中文乱码：根目录config文件

    language: zh-Hans或者language: zh-CN

    （取决于你的主题theme目录下的language目录下有zh-Hans.yml还是zh-CN.yml）

  • 头像/图标

    • 存放位置：

      头像/图标图片的存放位置是/themes/yilia/source/下任意位置，可以自己新建一个文件夹存放，我存放在assets文件夹下。

    • 配置设置：

      配置文件为/themes/yilia/_config.yml。设置头像为配置文件中avatar一项，设置图标为配置文件中favicon一项，设置路径的根目录为/themes/yilia/source/。例如，我的头像存放的地址是/themes/yilia/source/assets/me.png，设置则为avatar: /assets/me.png。（图标同理）
