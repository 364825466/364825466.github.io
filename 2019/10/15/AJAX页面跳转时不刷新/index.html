AJAX页面跳转时不刷新

做项目时订单页面和订单详情页面跳转时，页面没有刷新，再手动刷新一下就都出来了

订单页跳转

<span class="list_cz">
      <a class="infoorder" href="localhost:8080/userhtml/orderdetail.html?orderId=${result.data[i].orderFormId}" >
      查看订单
      </a>
 </span>

订单详情ajax

<script>
				// 从url上拿参数
               $.getUrlParam = function (name) {
                   var reg = new RegExp("(^|&)" + name + "=([^&]*)(&|$)");
                   var r = window.location.search.substr(1).match(reg);
                   if (r != null) return unescape(r[2]); return null;
               }

               var orderId = $.getUrlParam('orderId')
            
                    $.ajax({
                        type:"POST",
                        url:"http://localhost:8080/order/selectOrderByOrderId",
                        data:{
                            orderId:orderId,
                        },
                        dataType:"json",

                        success:function (result) {
                            if(result.code==1){
                                alert(result.message)
                            } else {

                                $("#allmoney").html(result.data.originPrice)
                                $("#orderid").html(result.data.orderFormNumber)
                                $("#orderstatus").html(result.data.orderFormStatus)
                                $("#hotelName").html(result.data.hotelName)
                                $("#hotelposition").html(result.data.hotelPositon)
                                $("#hotelPhoneNumber").html(result.data.hotelPhoneNumber)
                                $("#ruzhuname").html(result.data.userName)
                                $("#userTel").html(result.data.userPhoneNumber)
                                $("#startTime").html(result.data.arriveDate)
                                $("#endTime").html(result.data.leaveDate)
                                $("#roomName").html(result.data.hotelRoomName)

                            }
                        }
                    })

       </script>

询问了大佬之后，发现是一个很小的问题

跳转的时候没有加http！

改了订单页url，加上http://

<span class="list_cz">
    <a class="infoorder" href="http://localhost:8080/userhtml/orderdetail.html?orderId=${result.data[i].orderFormId}" >
    查看订单
    </a>
</span>
